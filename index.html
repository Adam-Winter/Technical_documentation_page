python introduction

A computer is composed of these parts:

  • The Central Processing Unit (CPU) is the part built to ask: what next? If a computer is rated at 3.0 Gigahertz, it means the CPU will ask this question three billion times per second.
  • The Main Memory stores information that the CPU needs in a hurry. It is nearly as fast as the CPU. The information stored in the main memory vanishes when the computer is turned off.
  • The Secondary Memory stores information even when the computer is off, but is much slower than the main memory. Examples include disk drives or flash memory.
  • Input and Output Devices are ways we interact with computers: screen, keyboard, mouse, mic, speaker…
  • The Network Connection retrieves data over a network, a very slow place to store and retrieve data that might not always be up. It’s is a slower and at times unreliable form of Secondary Memory.

A programmer writes programs to orchestrate these resources to solve problems and analyze data, telling the CPU to use the main memory, secondary memory, network, or the input/output devices. To do so, a propgramming language is used. Python has a set of reserved words : 

                    
                        and del from None True as elif global nonlocal try assert else if not while break except import or with class False in pass yield continue finally is raise async def for lambda return await

variables and expressions

Variable names must start with a letter or an underscore (but variable names starting with an underscore are usually used for another type of variables than we’re concerned with right now), they could contain letters, numbers and underscores and they’re case sensitive. Unlike in JavaScript, python doesn’t have a specific keyword to declare a variable.

When you declare a variable, python knows automatically what type it is. Some operations are prohibited on some variable types, such as adding a number to a string (unlike JavaScript, python doesn’t convert the number to a string and concatenate them). The type() function returns a variable’s type: 

                    
                        x = 3
                        type(x) <# class ‘int’>

conditional execution

The if, elif, and else keywords let us build conditional structures. Here, the two dots following these keywords and the indentations after them are crucial for proper code execution, because, unlike JavaScript, curly braces don’t exist to limit the code inside each condition.

  • Comparison operators are used to make the conditional decisions: ==, >=, <=, >, <, and !=. These ones are the same as in JavaScript.
  • There are also strict comparison operators, is and is not (a bit like === and !== in JavaScript) that are stronger than the standard == and != , meaning that not only do they seek equality in value but also in data type. This way, 0 == 0.0 is True (same value), but 0 is 0.0 is False (different types: one is int and the other float).

More advanced conditional structures exist: the try/except structure allows to try instructions that we anticipate to run badly, and offers an alternative if it does to avoiding a traceback.

                    
                        hours = input(“Enter working hours: “)
                        rate = input(“Enter rate: “)
                        try:
                            hours = float(hours)
                            rate = float(rate)
                        except:
                            print(“Error, please enter numeric input”)
                            quit()
                        pay = hours * rate
                        print(pay)

functions

Whether we define it or it’s already built-in, a function is a bunch of reusable code that takes arguments as input, does some computation with it, and returns an output value. If we want to define a function ourselves, we use the def keyword. We call/invoke a function by its name (just like built-in functions).

We use the def keyword to create a function, followed by the function name and eventual parameters in parentheses. The body of the function is indented. This defines the functions but does not execute it, only stores it and remembers it for future calls.

	                        if lang == ”de”
		                        return “Guten Tag!”

		                        return “Salut!”
	                        else:
		                        return “Hello!”
                        print(greet(“fr”), “Jean-Michel”)

strings

Strings are a sequence of characters between quotes. We can concatenate strings using +, we cannot concatenate a string and a number, but we can convert a string to a number using the int() function if it contains numeric characters only. User input is always a string unless we convert it if possible.

We access a character in a string using its index between square brackets. Indexing is zero-based. It could be an expression that is computed.
